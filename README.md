图床|TUCHUANG-您的网络图片存储库，基于百度云存储构建。
---------------------------------------------------------------
使用说明：
1、打开conf.inc.php，修改里面的常量值，改为自己申请到的appkey与密钥。
2、默认集成了百度、腾讯微博、新浪微博登录，如果不需要，可以自行搜索相关内容去除。
3、在百度云管理后台创建一个数据库，将/sql/init.sql中的内容导入执行。
4、开始享受吧。
